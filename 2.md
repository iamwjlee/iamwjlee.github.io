


# Network Communication Protocol(ODT,POS)

## 🎲 Server(POS) and Client(ODT) Link Sequence
``` javascript
const SOH = 0x01
const STX = 0x02
const ETX = 0x03
const EOT = 0x04
const ENQ = 0x05
const ACK = 0x06
const BS = 0x08

const LinkSequence = [0xf0, 0xf0, 0x05]
var pumpId = "04" //fixed 2bytes 
var dataLen = "0020" //fixed 4bytes
var Data=[]  //Ascii or Hex  
DATA=["04", ...Data]   //  pumpId+Data
DATA=["04", "0020", ...Data] //pumpId+dataLen+data

CommandFormat = [SOH,"ST",STX, ...DATA,ETX,LRC,EOT] //example Command="ST"
```

| Command | Description   
| ---- | ----   
| Server send **ENQ** | LinkSequence + ENQ(0x05) 
| Client reply **OF** | CommandFormt[01,4F,46,02,03,10,04] 

## 🎲 Server(POS) and Client(ODT) Communication Type

| Command | Description   
| ---- | ----   
| POS reply **ACK** | LinkSequence  + ACK(0x06) 
| POS reply **DATAACK** | LinkSequence + BS(0x08) 
| POS send **CMD** | LinkSequence + CommandFormat 
| ODT reply **ACK** | ACK 
| ODT send **CMD** | CommandFormat

## 🎲 Command( ODT <-> POS ) Tranfer Examples

| ST | Description |
| ---- | ---- |
| POS send ENQ | LinkSequence + ENQ 
| ODT send ST | CommandFormt[SOH,"ST",STX, ...DATA,ETX,LRC,EOT]
| POS reply ST | LinkSequence + CommandFormt[SOH,"ST",STX, ...DATA,ETX, LRC,EOT]
| ODT reply ACK | ACK


| PP(Charging) | Description |
| ---- | ---- |
| POS send ENQ | LinkSequence + ENQ
| ODT send PP | CommandFormt[SOH,"PP",STX, ...DATA,ETX,LRC,EOT]


| TM(Charged) | Description |
| ---- | ---- |
| POS send ENQ | LinkSequence + ENQ
| ODT send TM | CommandFormt[SOH,"TM",STX, ...DATA,ETX,LRC,EOT]
| POS reply ACK | LinkSequence + BS(0x08)


| Keydownload | Description |
| ---- | ---- |
| POS send ENQ | LinkSequence + ENQ 
| ODT send I0 | CommandFormt[SOH,"I0",STX, ...DATA,ETX,LRC,EOT]
| POS reply ACK | LinkSequence + ACK  
| POS reply NG | LinkSequence + CommandFormt[SOH,"NG",STX,"Failure",ETX,LRC,EOT]


| Inquire Point| Description |
| ---- | ---- |
|POS send ENQ| LinkSequence + ENQ
|ODT send KG|  CommandFormt[SOH,"KG",STX, ...DATA,ETX,LRC,EOT]
|POS reply ACK| LinkSequence + ACK
|POS reply OK|  LinkSequence + CommandFormt[SOH,"OK",STX,"Completion",ETX,LRC,EOT]
|POS reply KG|  LinkSequence + CommandFormt[SOH,"KG",STX, ...DATA,ETX,LRC,EOT]


| Credit Payment| Description 
| ---- | ---- 
|POS send ENQ| LinkSequence + ENQ
|ODT send SX| CommandFormt[SOH,"SX",STX, ...DATA,ETX,LRC,EOT]
|POS reply ACK| LinkSequence + ACK
|POS reply OK|  LinkSequence + CommandFormt[SOH,"OK",STX,"Completion Approval:1234", ETX,LRC,EOT]
